# Dotfiles

My Arch Linux dotfiles managed with [GNU Stow](https://www.gnu.org/software/stow/).

## Usage

```bash
git clone https://github.com/yourusername/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap.sh
```

## Tools Covered

- Bash
- Zsh + Powerlevel10k
- Git
- Neovim
- Alacritty
- i3 WM
- Tmux
